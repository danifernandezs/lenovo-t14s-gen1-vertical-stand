# 3D Printed vertical stand

3D Printed Vertical Stand for the Lenovo T14S Gen1

## Printer used

**Printer:** bq Prusa i3 hephestos<br>
**Rafts:** No<br>
**Supports:** No<br>
**Resolution:** 0.2mm<br>
**Infill:** 8%<br>
**Filament:** Sakata PLA
